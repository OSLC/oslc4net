# Getting Started

## Installation

OSLC4Net is available as NuGet packages. The main packages are:

- **OSLC4Net.Core** - Core OSLC functionality and annotations
- **OSLC4Net.Client** - Client library for consuming OSLC resources
- **OSLC4Net.Core.DotNetRdfProvider** - RDF provider using dotNetRDF
- **OSLC4Net.Core.JsonProvider** - JSON-LD provider

### Installing via NuGet

```bash
dotnet add package OSLC4Net.Client
```

Or via Package Manager Console:

```powershell
Install-Package OSLC4Net.Client
```

## Prerequisites

- .NET 6 or later runtime
- For development: .NET 8 SDK or later
- Libraries target .NET Standard 2.0/2.1
